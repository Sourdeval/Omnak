Points

Plus vos atouts sont avantageants, plus ils coûtent de points. Votre personnage est limité à 3 points.
C'est au MJ de décider combien vaut de points chacun de vos atouts. Voici une liste pour vous donner une idée des points :

Atout décisif améliorant grandement le personnage : 3 points
Atout important améliorant une partie du personnage : 2 points
Atout mineur, ouvrant une discipline secondaire ou donnant un bonus occasionnel : 1 point
Atout neutre, parfois bon parfois mauvais : 0 point
Atout qui donne un malus dans un aspect esquivable par le personnage : 0 point
Atout qui malus occasionnellement ou qui bloque un aspect secondaire du personnage : -1 point
Atout qui malus une partie du personnage : -2 points
Atout décisif qui malus grandement le personnage : -3 points

Si l'Atout bonus peut devenir un malus occasionnellement : -1 point
Si l'Atout malus peut devenir un bonus occasionnellement : +1 point
